# percentage
## Clone
```
git clone https://github.com/arfanamd/percentage.git
```
## Compile
```
cd percentage && cc -Wall -g percentage.c -o percentage && mv percentage /usr/bin
```
## Execute
percentage `[-ndf] <compare> <to>`

  Options:
  
    * -n[ormal]
    
    * -d[ifference]
    
    * -f[ractional]

## Man page
```
man ~/percentage/percentage.1.gz
```
